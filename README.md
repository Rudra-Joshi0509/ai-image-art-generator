# 🎨 AI Image Art Generator

A web-based application that converts images into artistic styles like sketch, cartoon, and edge detection using image processing.

---

## 🚀 Features

- Upload single or multiple images
- Convert images into:
  - ✏️ Pencil Sketch
  - 🎨 Cartoon Effect
  - 🖤 Black & White
  - 🔍 Edge Detection
- Real-time processing
- Download processed images
- Clean and interactive UI

---

## 🧠 Technologies Used

- Python
- OpenCV
- NumPy
- Streamlit
- Pillow
